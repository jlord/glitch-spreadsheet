How to get your spreadsheet key

In Google Sheets (Spreadsheets) create a new spreadsheet. Click File then Publish to the Web. In the new window that opens, click Publish for the entire document. Now your spreadsheet will be reachable (not editable) and a URL is provided. In that URL there is a set of uppercase letters and numbers between two forward slashes. That is your spreadsheet key. Copy it to use here!
